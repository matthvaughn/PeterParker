HTML: Navigating Folder Structure

Folder and file structure is probably the trickiest concept for first-time web developers to understand, so lets practice some more. Consider the file structure in Figure 21.1 and answer the questions listed below the image. You may want to cover the answers with your hand or a sheet of paper to see if you can figure them out on your own. What should the attribute/value pair look like to link from?

Figure 21.1: File Structure

1. The index.html page in the main WebFolder to the movies.html page in the same folder?

Answer: href="movies.html"

2. The selected index.html page in the test folder to the resume.html page in the homework folder?

Answer: href="../resume.html"

3. The index.html page in the WebFolder to the up.jpg image in the images folder?

Answer: src="images/up.jpg"

4. The selected index.html page in the test folder to the bullet.gif image in the menu folder?

Answer: src="../../images/menu/bullet.gif"

You have now learned some basic HTML elements and how to link between files. The next chapter will teach you how to begin styling these elements using Cascading Style Sheets. You will use the files you created in this chapter in the next chapter.
